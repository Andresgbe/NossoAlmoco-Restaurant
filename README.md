# NossoAlmoco-Restaurant

This version was made in Java, originally it was developed in C++.

This project was made to be presented in a course at the Universidad Católica Andrés Bello (UCAB).

It is a virtual restaurant in which you must register as a customer or administrator to be able to use all the functions. If you are a client, you will have the option to choose dishes and food contours, you will also have suggestions and the option to save your favorite dishes no matter if you close the program.

The administrator can change the dishes of the day and contours, as well as give recommendations to customers.

There are other functions you can perform in the program.
